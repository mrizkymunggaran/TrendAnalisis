MADINA - Sebuah mobil pick up tiba-tiba terbakar setelah baru selesai mengisi premium di sebuah SPBU di Mandailing Natal, Sumatera Utara. Sang sopir yang terkena sambaran api menderita luka bakar serius. Sementara polisi masih terus menyelidiki penyebab pasti kebakaran mobil tersebut.     Dari rekaman gambar amatir ini terlihat kepanikan terjadi di SPBU setelah sebuah mobil pick up yang baru selesai mengisi premium tiba-tiba saja terbakar di lokasi pelataran sebuah SPBU di Mandailing Natal, Sumatera Utara, Sabtu (19/5/2013) malam.   Diduga, mobil terbakar akibat adanya percikan api dari kabel sistem kelistrikan mobil sesaat sang sopir menghidupkan mesin setelah selesai mengisi premium.   Petugas SPBU di bantu warga segera memadamkan api agar tak merembet ke pompa SPBU dengan menggunakan sistem pemadam api milik SPBU.   Api tak berapa langsung padam namun sang sopir pick up naas tersebut menderita luka bakar akibat tersambar api.   sang sopir bernama Sobar (35), warga Desa Padang Laru dilarikan ke rumah sakit untuk mendapat pertolongan. Sobar menderita luka bakar hingga 40 persen.   Menurut Sobar, ia tak menyadari kebakaran terjadi karena tiba-tiba jerigen berisi premium yang ia bawa meledak hingga membakar mobil miliknya.   Hingga kini, polisi masih menyelidiki kasus ini dengan meminta keterangan dari sopir pick dan petugas SPBU. (Ahmad Husein Lubis/Sindo TV)
