Jakarta (ANTARA News) - Piranti lunak sistem operasi open source Ubuntu versi 13.04 atau disebut Raring Ringtail telah tersedia di situs resmi Ubuntu baik untuk komputer personal ataupun komputer server. File unduhan OS Ubuntu 13.04 untuk komputer personal membutuhkan kapasitas penyimpanan data digital minimal 348 mega bit, tersedia untuk PC berprosesor Intel x86, PC berarsitektur prosesor AMD64 atau EM64T (64 bit), dan komputer Mac berarsitektur prosesor AMD64 atau EM64T (64 bit). Sementara, file Ubuntu 13.04 untuk komputer server juga terbagi dalam tiga kategori yang sama seperti komputer personal. Ubuntu, di situsnya, mengakui belum menyediakan file unduhan untuk komputer berarsitektur prosesor ARM. Tapi, tetap menyediakan file-file unduhan untuk perangkat keras berbasis Intel dan AMD di BitTorrent. Media teknologi Geek melaporkan OS terbaru Ubuntu itu mempunyai keunggulan kerja sistem lebih cepat, tapi dengan penggunaan ruang penyimpanan digital lebih rendah. \Ada juga pengembangan tampilan pengguna antar-muka (user interface), termasuk Ubuntu One yang mempunyai menu sinkronisasi untuk mengontrol aplikasi-aplikasi apa saja yang dapat mengakses akun sosial Anda di satu tempat,\" sebut Matthew Humphries dari Geek. Pengguna juga membuka beberapa aplikasi sekaligus (multi-tasking) dan menggeser jendela aplikasi-aplikasi itu dengan tombol gulir (scroll) pada tetikus (mouse). Bagi calon pengguna personal, Ubuntu dapat dipasang di mesin virtual untuk uji coba sebelum mengunduh gratis dan memasang secara permanen di komputer. \"Jika Anda ingin versi Ubuntu yang dapat dipakai sebagai sistem operasi di perangkat tablet dan ponsel pintar, Anda harus menunggu hingga versi 13.10 yang akan hadir pada akhir Oktober mendatang,\" sebut Humphries. Namun, Ubuntu 13.04 mendukung para pengembang yang ingin memasang sistem operasi itu di perangkat-perangkat seluler dengan menyediakan paket pengembangan sistem (SDK) Developer Preview."
