TEMPO.CO, Jakarta- Tim putra Surabaya Samator dan tim putri Gresik Petrokimia tampil sebagai pemenang ketiga kompetisi bola voli Proliga 2013 setelah mengungguli lawan-lawan mereka di Istora Gelora Bung Karno, Senayan, Jakarta, Ahad, 5 Mei 2013. Tim bola voli putra Surabaya Samator menang tipis atas Jakarta Sananta Indocement 3-2. Sedangkan di bagian putri, tim Gresik Petrokimia juga menang tipis 3-2 atas regu Jakarta Elektrik. Pada pertandingan tersebut tim putra Samator dan Petrokimia Gresik saling kejar dalam perolehan angka. Namun menjelang paruh akhir pertandingan set pertama, tim Samator dapat mengendalikan permainan dan memenangi set pertama 25-20. Di set kedua, Sananta mencoba tampil menekan. Hingga pertengahan pertandingan, Sananta terus menekan dan menguasai permainan. Para Samator kesulitan meladeni bola-bola cepat dan tekanan para pemain Sananta. Set ini akhirnya dimenangi Sananta 25-17. Memasuki set ketiga Samator melesat dalam pengumpulan angka dengan memanfaatkan kesalahan lawan saat service. Di tengah laga Sananta mencoba mengejar ketertinggalannya, namun selisih sudah terlalu jauh dan akhirnya kalah dari Samator 25-20. Pada set keempat Sananta mencoba bangkit dan kembali tampil menekan sehingga menang 28-26. Skor berubah menjadi imbang 2-2. Sayangnya pada set kelima penentuan, Sananta tak mampu mempertahankan konsistensi penampilannya dan kalah dari Samator 15-13.
