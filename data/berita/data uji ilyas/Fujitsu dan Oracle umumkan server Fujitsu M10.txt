Jakarta (ANTARA News) - Melanjutkan kemitraan yang telah berlangsung lebih dari 20 tahun, Fujitsu dan Oracle mengumumkan ketersediaan rangkaian server Fujitsu M10 yang menggunakan prosesor SPARC64 X 16. Rangkaian server Fujitsu M10 itu dapat ditingkatkan secara dinamis dari 1 hingga 64 prosesor menggunakan arsitektur modular dengan core-level CPU Activation, kata Fujitsu dalam pernyataannya, Rabu. Arsitektur modular dengan core-level CPU Activation memungkinkan pelanggan korporat bisa dengan mudah menambahkan komponen secara bertahap untuk memenuhi kebutuhan aktivitas komputasi yang selalu berubah dan meningkat. Fujitsu mengatakan, dengan teknologi mutakhir termasuk Software on Chip dan Liquid Loop Cooling, server Fujitsu M10 menyediakan konfigurasi sistem yang sangat fleksibel dengan partisi fisik serta teknologi virtualisasi yang tersedia secara built in. Konfigurasi sistem dengan partisi fisik serta teknologi virtualisasi dapat dilakukan tanpa biaya melalui Oracle VM Server for SPARC dan Oracle Solaris Zones. Jajaran server Fujitsu M10 sudah mendukung baik sistem operasi Oracle Solaris 10 maupun 11, dan dengan Oracle Solaris Guarantee Program bisa kompatibel dengan semua server SPARC yang ada sekarang maupun versi sebelumnya, kata Fujitsu.
