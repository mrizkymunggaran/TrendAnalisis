Metrotvnews.com, Madrid: Rafael Nadal bangkit dari kekalahan di set pertama untuk mengalahkan David Ferrer 4-6, 7-6, dan 6-0 di babak perempat final turnamen tenis Madrid Terbuka. Kemenangan itu membuka peluang bagi Nadal untuk mencapai final ketujuhnya secara beruntun pascapulih dari cedera. Dalam pertemuan antara dua petenis papan atas Spanyol itu, permainan baseline Ferrer sempat menyulitkan Nadal. Namun, Nadal bangkit di set kedua untuk memaksakan laga diselesaikan lewat set ketiga. Di set penentuan itu, Nadal memenangkan enam game secara beruntun dan mengalahkan Ferrer. \Saya sangat senang bisa mencapai babak semi final. Apalagi melihat hebatnya permainan David,\" aku Nadal. "
