TEMPO.CO, Jakarta-Dell mulai menggarap pasar android dengan meluncurkan Project Ophelia bulan depan. USB dapat dihubungkan ke monitor atau perangkat lainnya sehingga pengguna dapat mengakses data tanpa PC, tablet, atau ponsel pintar. USB tersebut dapat memuat data pribadi maupun perusahaan. Gadget juga dapat terhubung melalui perangkat lunak Citrix, Microsoft, dan VMware. Kemampuan Project Ophelia dapat mengatur keamanan melalui perangkat individu, sehingga data dapat dihapus jika pengguna mengkhawatirkan adanya pencurian data. Pengguna juga dapat menggunakan cloud client manager untuk mengawasi cara kerja Ophelia ketika sedang digunakan. Ophelia dapat terhubung dengan koneksi Wi-Fi dan Bluetooth, serta dapat dihubungkan ke keyboard, mouse, dan dapat menjalankan sistem operasi berbasis android 4.0.
