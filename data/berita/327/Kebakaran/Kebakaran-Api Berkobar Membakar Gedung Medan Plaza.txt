MEDAN, KOMPAS.com - Gedung Medan Plaza yang berada di Jalan Iskandar Muda, Medan, Sumatera Utara, ludes terbakar. Api melalap pusat perbelanjaan itu sejak Sabtu (22/8/2015) dinihari, dan masih berkobar hingga pagi ini. Sejauh ini, belum diperoleh informasi mengenai penyebab kebakaran, maupun asal api. Di lokasi kebakaran terlihat api berkobar hebat dan ratusan warga berkerumun untuk menonton peristiwa ini. Kepadatan warga memaksa petugas meminta warga menjauh hingga jarak 100 meter dari lokasi kebakaram. Di jarak 100 meter itupun panas api sudah sangat terasa.
