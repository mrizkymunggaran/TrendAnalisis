MEDAN - Hingga pagi ini, api masih menyala di beberapa titik di dalam gedung Medan Plaza, Jalan Iskandar Muda, Medan. Petugas pemadam kebakaran Pemerintah Kota Medan kesulitan untuk menjangkau api yang masih menyala di dalam gedung karena eskalator maupun tangga gedung tidak bisa lagi digunakan. Petugas terpaksa menggunakan tangga naik ke lantaiï¿½ 2 dan 3 gedung plaza untuk melihat dan menyiram api. Sementara untuk menjangkau ke lantai 4 hingga lantai 6 gedung, petugas pemadam kebakaran menggunakan mobil tangga. Setelah 32 jam terbakar gedung medan plaza masih menyisakan beberapa percikan api dan tumpukan sampah yang masih menyala. \Kita akan berupaya untuk memadamkan seluruh titik api pada hari ini,\" kata Komandan Regu Pemadam Kebakaran Kota Medan. Petugas laboratorium forensik Polri belum dapat melakukan olah tempat kejadian perkara di lokasi kejadian, karena api di dalam gedung belum seluruh nya padam."
