SUKABUMI - Bangunan yang terdiri atas 25 kamar milik sebuah peternakan ayam di Cikundul, Kecamatan Lembursitu, Kota Sukabumi, Jawa Barat hangus terbakar pada Sabtu 15 Agustus 2015 malam. Peristiwa nahas tersebut diduga akibat korsleting listrik. Berdasarkan informasi yang diperoleh, api kali pertama terlihat dari salah satu kamar yang terletak di tengah bangunan. Saat peristiwa terjadi, bangunan yang merupakan mess 75 pekerja peternakan itu dalam kondisi sepi. Kobaran api dengan mudah semakin membesar karena embusan angin, Alhasil api menjalar cepat ke setiap bangunan. Dalam waktu singkat api melalap seluruh bangunan mess. Warga bersama penghuni mess kesulitan mematikan kobaran api. Api mulai berhasil dipadamkan setelah enam unit mobil pemdam kebakaran datang. Bencana kebakaran yang berlangsung selama lebih dari satu jam ini hanya menyisakan puing bangunan yang hampir seluruhnya menjadi arang. Andi, salah seorang warga mengatakan kepulan asap kali pertama terlihat dari tengah bangunan mess. Lantaran lokasinya berada jauh dari permukiman penduduk, kebakaran baru diketahui setelah api membesar. \Sepertinya api susah sekali dijinakan, sampai harus berlangsung selam satu jam lebih,\" ujar Andi. Kapolsek Lembursitu AKP Supardi menerangkan tidak ada korban jiwa dalam kejadian kebakaran di lokasi mess peternakan. Hasil pendataan sementara kerugian akibat bencana tersebut diperkirakan mencapai ratusan juta rupiah. \"Bukan cuma bangunan saja, tapi harta benda milik penghuni turut terbakar. Diduga api berasal dari korsleting listrik pada salah satu kamar,\" tuturnya."
