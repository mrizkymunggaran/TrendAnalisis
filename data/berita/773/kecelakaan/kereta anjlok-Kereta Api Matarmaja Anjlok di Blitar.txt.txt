JAKARTA - Kereta Api Matarmaja jurusan Pasar Senen Jakarta menuju Malang, anjlok di Kilometer 115, Kecamatan Garum, Kabupaten Blitar, Jawa Timur, Senin (24/8/2015) pagi. Menurut petugas jaga Polsek Garum Bripka Sutardi, kereta tersebut anjlok sekitar pukul 07.30 WIB. \Bapak Kapolsek (AKP Sudarto) sudah ke lokasi sejak mendapat laporan. Sejauh ini tidak ada laporan korban,\" ujar Bripka Sutardi saat dihubungi Sindonews, Senin (24/8/2015). Sementara, seperti dikutip dari Okezone.com, kereta terseret sejauh 300 meter, membuat sekira 1.000 penumpang di dalam 10 gerbong berhamburan menyelamatkan diri. Mereka panik mendengar suara gesekan keras dan kereta yang tiba-tiba miring. Kereta api Matarmaja tersebut anjlok sekira pukul 07.00 WIB. Kecelakaan itu mengakibatkan kurang lebih 400 bantalen rel lepas, karena kereta keluar dari jalur. \"Kaget sekali karena tiba-tiba kereta mengerem dan miring. Terpenting menyelamatkan diri,\" ujar Deny Firmansyah, salah satu penumpang. Para penumpang yang naik kereta api itu terpaksa beralih menggunakan bus menuju Malang. Selain penumpang Kereta Matarmaja, penumpang Kereta Api Gajayana rute Jakarta-Malang di Stasiun Blitar, juga terkena imbas. Kereta mereka tidak bisa melanjutkan perjalanan dan mereka diantarkan ke Malang menggunakan bus."
